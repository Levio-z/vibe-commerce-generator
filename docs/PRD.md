# 电商内容智能生成工具 - PRD

## 1. 项目概述

### 1.1 项目背景
当前运营团队为每个商品撰写图文内容需要从零开始，耗时耗力。本项目旨在构建一个智能内容生成工具，帮助运营批量生成商品图文草稿，将工作模式从"白手起稿"转变为"筛选+微调"。

### 1.2 项目目标
- 第一阶段：实现单个商品的图文内容生成验证
- 支持商品基础信息录入、图片上传、参考素材上传
- 生成包含基础卖点的主图草稿 + 结构化标题 + 卖点文案

## 2. 功能需求

### 2.1 商品信息录入模块

**输入方式：**
- 在线表单录入
- Excel模板批量导入（后续版本）

**字段列表：**
| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 商品名称 | 文本 | 是 | |
| 商品类目 | 下拉选择 | 是 | 如：服装、数码、家居等 |
| 品牌 | 文本 | 否 | |
| 材质 | 文本 | 否 | |
| 尺寸 | 文本 | 否 | |
| 颜色 | 文本 | 否 | |
| 适用人群 | 下拉选择 | 否 | 如：男士、女士、儿童、通用 |
| 其他卖点 | 多行文本 | 否 | 运营补充的额外卖点 |

### 2.2 图片上传模块

**支持上传：**
- 商品主图（白底图/场景图）
- 参考素材（历史爆款截图、竞品参考图）

**功能特性：**
- 拖拽上传
- 图片预览
- 支持多图上传（最多5张参考图）
- 支持删除已上传图片

### 2.3 内容生成模块

**生成触发：**
- 点击"生成内容"按钮
- 校验必填字段

**输出内容：**
1. **标题生成**
   - 结构：品牌 + 类目 + 核心特性 + 适用场景
   - 包含核心关键词
   - 字数控制：20-30字

2. **卖点文案**
   - 1-2句精炼卖点
   - 基于商品属性自动提取

3. **主图草稿预览**
   - 在商品图上叠加文字卖点
   - 支持简单的布局选择（居中、底部、顶部）

### 2.4 结果展示与导出

**展示：**
- 分栏展示：左侧输入、右侧生成结果
- 实时预览主图效果

**操作：**
- 一键复制标题和文案
- 下载主图草稿
- 保存到素材库（后续版本）
- 重新生成（可调整参数）

## 3. 技术实现方案

### 3.1 技术栈
- **前端框架：** React + TypeScript
- **构建工具：** Vite
- **UI组件库：** Ant Design
- **图片处理：** HTML5 Canvas
- **AI生成模拟：** 前端规则引擎（演示用，后续接入真实API）

### 3.2 项目结构
```
src/
├── components/
│   ├── ProductForm.tsx      # 商品信息表单
│   ├── ImageUploader.tsx    # 图片上传组件
│   ├── ResultPanel.tsx      # 结果展示面板
│   └── MainPreview.tsx      # 主图预览组件
├── hooks/
│   └── useContentGenerator.ts # 内容生成逻辑
├── utils/
│   ├── canvasUtils.ts       # Canvas图片处理工具
│   └── titleGenerator.ts    # 标题生成规则
├── types/
│   └── index.ts             # 类型定义
├── App.tsx
└── main.tsx
```

### 3.3 核心算法说明

**标题生成规则：**
```
模板1：{品牌} {材质} {类目} {颜色} {适用人群}款
模板2：{品牌} {核心特性} {类目} - {尺寸}
模板3：{适用人群} {类目} | {材质} {颜色} | {品牌}
```

**卖点提取规则：**
- 材质 → "优质{材质}，舒适耐用"
- 品牌 → "{品牌}品质保证"
- 适用人群 → "专为{适用人群}设计"

## 6. 验收标准

- 能够完整录入商品所有基础信息
- 能够上传商品图片并预览
- 点击生成后3秒内显示结果
- 生成的标题包含至少3个核心属性
- 主图预览正确显示图片和文字叠加效果
- 支持复制生成的文案内容
- 支持下载生成的主图
