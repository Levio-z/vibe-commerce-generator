
# 电商内容智能生成工具 - 代码生成提示词

## 项目概述
创建一个单页面React应用，用于验证电商商品图文内容的智能生成。帮助运营从"白手起稿"转变为"筛选+微调"的工作模式。

## 技术栈要求
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI样式**: Tailwind CSS v3
- **图标**: lucide-react
- **不需要**: 无需后端API：只需前端实现，所有逻辑在浏览器端完成

interface ProductInfo {

## 功能实现要求

### 1. 商品表单组件
- 必填字段标记（*）验证
- 类目下拉选项：['服装', '数码', '家居', '美妆', '食品', '箱包', '家电', '其他']
- 适用人群下拉选项：['男士', '女士', '儿童', '通用']
- 实时表单数据绑定

### 2. 图片上传组件
- 支持点击选择文件
- 支持拖拽上传
- 图片预览缩略图显示
- 删除已上传图片功能
- 限制：商品主图限制1张，参考素材最多5张

### 3. 内容生成引擎（前端规则引擎）

**标题生成规则：**
- 模板1：`{brand} {material} {category} {color} {targetAudience}款
- 模板2：`{brand} {核心特性词} {category} - {size}`
- 模板3：`{targetAudience} {category} | {material} {color} | {brand}`
- 随机选择模板，确保标题长度20-30字

**卖点生成规则：**
- 材质 → "优质{material}，舒适耐用，触感细腻"
- 品牌 → "{brand}品质保证，正品保障"
- 适用人群 → "专为{targetAudience}设计，贴合需求"
- 颜色 → "{color}时尚配色，百搭耐看"
- 尺寸 → "多尺寸可选：{size}"
- 生成1-2条卖点

### 4. Canvas主图生成
- 将上传的商品图片绘制到Canvas
- 根据选择的布局位置（居中/底部/顶部）叠加文字
- 文字样式：白色+黑色描边，字号适中
- 支持导出为PNG

### 5. 结果展示与操作
- 实时预览生成的主图
- 显示标题和卖点文案展示
- 一键复制文案到剪贴板
- 下载主图到本地
- 重新生成功能（保留输入内容）
- 布局切换实时更新预览

## 交互细节要求

1. **初始状态：右侧显示"请填写商品信息并上传图片后点击生成"占位

2. **生成中状态：按钮显示loading状态

3. **生成完成：右侧平滑过渡显示结果

4. **响应式：适配桌面端，最小宽度1200px

5. **用户体验优化：**
   - 表单输入即时反馈
   - 图片上传进度提示
   - 复制成功提示
   - 下载loading状态
