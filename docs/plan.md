# 电商内容智能生成工具 - 开发计划

## 🎯 项目目标
创建一个单页面应用，实现电商商品图文内容智能生成的核心玩法验证。

- ✅ 参考素材上传
- ✅ 智能标题生成（基于规则引擎）
- ✅ 卖点文案生成（1-2条）
- ✅ Canvas主图草稿生成（文字叠加）

## 📝 核心算法说明

### 标题生成模板
1. `{品牌} {材质} {类目} {颜色} {适用人群}款`
2. `{品牌} {特性词} {类目} - {尺寸}`
3. `{适用人群} {类目} | {材质} {颜色} | {品牌}`

### 卖点生成规则
- 材质 → "优质{material}，舒适耐用"
- 品牌 → "{brand}品质保证"
- 适用人群 → "专为{targetAudience}设计"

### Canvas渲染
- 图片按比例缩放至800x800
- 文字：白色填充 + 3px黑色描边
- 支持三种布局位置
